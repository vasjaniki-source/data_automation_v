# Руководство пользователя

1. **Вкладка Загрузка**: Выберите файл или введите URL API. Нажмите "Загрузить".
2. **Вкладка База данных**: Настройте параметры PostgreSQL. Используйте "Проверить подключение".
3. **Панель управления**: Кнопки "Валидация", "Очистка" и "Анализ" должны нажиматься последовательно.
4. **Лог событий**: В нижней части окна отображается история всех действий и ошибок.
