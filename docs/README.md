# Автоматический анализатор данных

Графический интерфейс для комплексного анализа данных с поддержкой различных источников ввода. Программа позволяет загружать, валидировать, очищать данные, выполнять аналитические операции и формировать отчёты в форматах PDF и Excel с возможностью отправки по email.

## Функциональные возможности

**Загрузка данных из источников:**
* CSV/Excel‑файлы;
* API (по URL);
* SQL‑запросы;
* PostgreSQL (с вводом хоста, порта, БД, логина и пароля).

**Анализ данных:**
* полный анализ набора данных;
* анализ временных рядов и декомпозиция;
* построение ML‑моделей;
* расчёт корреляционной матрицы;
* визуализация распределений (гистограммы).

**Обработка данных:**
* валидация структуры и типов данных;
* очистка от пропусков, дубликатов и аномалий.

**Отчётность:**
* генерация PDF‑отчётов с результатами анализа;
* экспорт в Excel с данными и аналитикой;
* отправка отчётов по email.

**Дополнительные функции:**
* история выполненных операций;
* автосохранение данных;
* тестирование соединения с БД;
* демо‑режим (загрузка тестовых данных);
* логирование действий и ошибок.

## Установка и запуск

### Требования
* Python 3.8 или выше;
* установленные библиотеки (см. `requirements.txt`).

### Установка зависимостей

* pip install -r requirements.txt

#### Запуск программы
* python scripts\run_gui.py
