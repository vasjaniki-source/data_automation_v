# data_automation_v
Автоматический анализатор данных  Графический интерфейс для комплексного анализа данных с поддержкой различных источников ввода. Программа позволяет загружать, валидировать, очищать данные, выполнять аналитические операции и формировать отчёты в форматах PDF и Excel с возможностью отправки по email.
